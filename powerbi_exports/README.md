# Power BI Data Export

Export corridor data to CSV files for Power BI analysis.

## Usage

```bash
npm run export:powerbi
```

## Output Files

- `stations.csv` - Station data with real-time metrics
- `trains.csv` - Train schedules and routes  
- `delay_summary.csv` - Hourly delay analysis

## Requirements

- Node.js 18+
- Running API server on localhost:3000
